Ласкаво просимо на мою сторінку

На цій сторінці ви знайдете посилання на мою галерею та список команд для роботи з терміналом Git та GitHub:

Список команд GitHub:

  • git init - ініціалізує Git репозиторій
  • git clone [URL] - клонує віддалений репозиторій
  • git add [файл] - додає файл до індексу
  • git commit -m "Коміт" - створює коміт
  • git push [віддалений репозиторій] [гілка] - надсилає зміни до віддаленого репозиторію
  • git pull [віддалений репозиторій] [гілка] - отримує оновлення з віддаленого репозиторію
  • git branch - переглядає список гілок
  • git checkout [гілка] - перемикається на іншу гілку
  • git merge [гілка] - зливає гілку з поточною гілкою
  • git status - переглядає статус репозиторію
